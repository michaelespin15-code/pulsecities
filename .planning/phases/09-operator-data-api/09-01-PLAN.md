    Create models/operators.py with two models: Operator and OperatorParcel.

    ```python
    """
    Operator registry and parcel membership tables.

    operators — one row per identified operator cluster (e.g. MTEK, PHANTOM CAPITAL).
                Slug is the stable public identifier used in URLs and API routes.
    operator_parcels — join table: one row per (operator, BBL) pair.
                       Backfilled from ownership_raw; refreshed nightly (future: FRESH-01).
    """

    from datetime import date
    from decimal import Decimal

    from sqlalchemy import Date, Float, ForeignKey, Index, Integer, Numeric, String, UniqueConstraint
    from sqlalchemy.dialects.postgresql import JSONB
    from sqlalchemy.orm import Mapped, mapped_column

    from models.base import Base, TimestampMixin


    class Operator(TimestampMixin, Base):
        __tablename__ = "operators"

        slug: Mapped[str] = mapped_column(String(100), nullable=False)
        operator_root: Mapped[str] = mapped_column(String(50), nullable=False)
        display_name: Mapped[str] = mapped_column(String(200), nullable=False)
        llc_entities: Mapped[list] = mapped_column(JSONB, nullable=False, default=list)
        total_properties: Mapped[int] = mapped_column(Integer, default=0)
        total_acquisitions: Mapped[int] = mapped_column(Integer, default=0)
        # Cached aggregates recomputed by backfill script
        borough_spread: Mapped[int | None] = mapped_column(Integer, nullable=True)
        highest_displacement_score: Mapped[float | None] = mapped_column(Float, nullable=True)

        __table_args__ = (
            UniqueConstraint("operator_root", name="uq_operators_root"),
            UniqueConstraint("slug", name="uq_operators_slug"),
            Index("idx_operators_slug", "slug"),
        )


    class OperatorParcel(TimestampMixin, Base):
        __tablename__ = "operator_parcels"

        operator_id: Mapped[int] = mapped_column(
            Integer, ForeignKey("operators.id", ondelete="CASCADE"), nullable=False
        )
        bbl: Mapped[str] = mapped_column(String(10), nullable=False)
        acquiring_entity: Mapped[str | None] = mapped_column(String(200), nullable=True)
        acquisition_date: Mapped[date | None] = mapped_column(Date, nullable=True)
        acquisition_price: Mapped[Decimal | None] = mapped_column(Numeric(15, 2), nullable=True)

        __table_args__ = (
            UniqueConstraint("operator_id", "bbl", name="uq_operator_parcels_op_bbl"),
            Index("idx_operator_parcels_operator_id", "operator_id"),
            Index("idx_operator_parcels_bbl", "bbl"),
        )
    ```

    After creating the file, add "from models.operators import Operator, OperatorParcel" to models/__init__.py if that file exists; if not, no action needed (Alembic reads Base.metadata via env.py which imports all models).

    **Part B — Backfill script**

    Create scripts/backfill_operators.py. The script:

    1. Reads scripts/operator_network_analysis.json (20 operators).
    2. For each operator, derive slug using this exact mapping applied first for the three named operators, then a generic rule for the rest:
       - MTEK → mtek-nyc
       - PHANTOM CAPITAL → phantom-capital
       - BREDIF → bredif
       - Generic rule for any other: operator_root.lower().replace(" ", "-").replace("_", "-")
    3. Upserts into operators table using ON CONFLICT (operator_root) DO UPDATE.
    4. After upserting the operator row, queries operator ID, then populates operator_parcels:
       ```sql
       SELECT DISTINCT ON (bbl)
           bbl,
           party_name_normalized AS acquiring_entity,
           doc_date AS acquisition_date,
           doc_amount AS acquisition_price
       FROM ownership_raw
       WHERE party_type = '2'
         AND party_name_normalized = ANY(:llc_names)
       ORDER BY bbl, doc_date DESC
       ```
       Bulk inserts with ON CONFLICT (operator_id, bbl) DO NOTHING to avoid duplicates.
    5. After all parcels are inserted, compute and UPDATE the cached aggregate columns:
       - borough_spread:
         ```sql
         SELECT COUNT(DISTINCT SUBSTRING(op.bbl, 1, 1))
         FROM operator_parcels op
         WHERE op.operator_id = :operator_id
         ```
       - highest_displacement_score:
         ```sql
         SELECT MAX(ds.score)
         FROM operator_parcels op
         JOIN parcels p ON p.bbl = op.bbl
         JOIN displacement_scores ds ON ds.zip_code = p.zip_code
         WHERE op.operator_id = :operator_id
         ```
    6. Commits after all 20 operators are processed.
    7. Prints a summary line: "Seeded N operators, M parcels total."

    Use models.database.get_scraper_db() context manager for the session. Import text from sqlalchemy.

    Slug derivation function (put at top of script):
    ```python
    _SLUG_OVERRIDES = {
        "MTEK": "mtek-nyc",
        "PHANTOM CAPITAL": "phantom-capital",
        "BREDIF": "bredif",
    }

    def _make_slug(operator_root: str) -> str:
        if operator_root in _SLUG_OVERRIDES:
            return _SLUG_OVERRIDES[operator_root]
        return operator_root.lower().replace(" ", "-").replace("_", "-")
    ```

    Run the backfill after creating it:
    ```bash
    cd /root/pulsecities && python scripts/backfill_operators.py
    ```
